# homebrew-agentrail

Homebrew tap for `agentrail`.

## Install

```bash
brew tap Latentik/agentrail
brew install agentrail
```

## How it is updated

This repository is updated automatically by the release workflow in `Latentik/agentrail`.
Tagged releases in the main project publish macOS binaries and then update `Formula/agentrail.rb` here.
